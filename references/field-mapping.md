# Field Mapping

Use this guidance when mapping rough notes into report sections.

## Common Mappings

- 完成、搞定、处理完、交付、提交、整理完 -> 今日完成 / 本周完成 / 已完成工作
- 推进、沟通、对齐、确认、同步、跟进 -> 关键进展 / 当前进度
- 卡住、延期、不确定、反馈不好、阻塞、缺资料 -> 问题风险 / 阻塞项
- 明天、下周、后续、准备、计划、继续 -> 下一步计划 / 明日计划
- 需要、麻烦、希望支持、待确认、等回复 -> 需要支持 / 待协调
- 数字、金额、比例、数量、增长、下降、完成率 -> 数据指标 / 量化结果
- 决定、确认、定了、口径、结论 -> 决策结论 / 已确认事项
- 复盘、原因、教训、经验 -> 复盘沉淀 / 经验总结

## Unclear Items

If an item could fit multiple sections, choose the most action-oriented section and add a note in `待确认项`.

If an item cannot be mapped with confidence, place it in `待确认项` instead of forcing it into the report.

## Empty Fields

If the template requires a field but no source material supports it, keep the field and write one of:

- `[暂无明确素材]`
- `[待补充]`
- `[需补充数据]`

Do not delete required fields from the user's template.
